OMNI CONVERTER

This system is designed to store directed acyclic graph (DAG) data with the ability to import from and export to a variety of file formats.

It supports XML, HTML, TXT, SQL, JSON, and CSV file formats, but be warned, each of these file formats has a very specific way of representing graph data. Before starting, carefully read the file format documentation and look at the sample upload files that were linked to in our initial email.

INSTRUCTIONS

You have been given access to this web based client, an executable Java archive, and a copy of the source code. Your task is to identify issues with the system as well as suggesting potential solutions.

There are no restrictions on how you should be testing the system, nor restrictions on the problems you should be looking for. However, some problems will be very difficult to identify without inspecting source code.

Access to the VM you have been assigned will be revoked after 5 days, but you can continue, running the server locally, for as long as you need. Once you are finished, you should email us your report.
